- **三级标签**：`L4 拔尖级`（跨模块高阶整合，含复杂参数和分类讨论）。
- **理由**：
  - **知识点深度交织**：至少 3 个知识点嵌套：

  - 符合 L4 典型特征：**旋转折叠叠加问题**（动态几何 + 对称性），见文档示例。
